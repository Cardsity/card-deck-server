Cardsity decks

A utility to manage decks for Cardsity.

What is Cardsity?
Cardsity is an open source Cards Against Humanity clone. It consists of three parts:

  • The game server, written in C++, maintained by Curve.
  • The game client, written in Vue, maintained by D3S0X.
  • The deck server, written using Django, maintained by ravi0li.
What can you do here?
On this page, you can create new decks as well as viewing other decks. You can use the decks you found here in the game client.
